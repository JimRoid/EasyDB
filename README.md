EasyDB
====================


### Description

Android SharePreference 

### Integration
   Including in your project
   -------------------------
   
   ```groovy
   maven { url 'https://github.com/JimRoid/EasyDB/raw/master/' }
   ```
   
   ```groovy
   compile 'com.easyapp.easydb:EasyDB:1.0'
   ```
   
### Sample
   
   ```java
        extend Model
   ```
   
   ```java
      EasyDB easy = new EasyDB();
   ```
